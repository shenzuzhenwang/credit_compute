# credit_compute

### 学分绩与平均绩点计算



一、添加课程成绩
  1.课程：
      课程的名称（可不填）
  2.学分：
      此课程对应的学分（可不为整数）
  3.成绩：
      此课程对应的成绩（绩点转换范围内）
  4.全选：
      当选择第一行的全选，所有课程都会选中，再次点击，则所有课程都未选中。
      当选择非第一行，则代表此行课程被选中，再次点击，则此课程未选中。
      当第一行全选按钮显示对勾时，代表全部课程选中；显示方块，代表部分课程选中；显示空，代表全部课程未选中。
      当非第一行按钮显示对勾时，代表课程选中；显示空，代表课程未选中。
